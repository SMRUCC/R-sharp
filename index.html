Shoal Programming Features

  • Memory pipeline in GCModeller

    • The memory pipeline was used in the GCModeller, which can save the times on the large data IO between the CLI tools when you are programming hybrids the .NET export API with the cmd shell in the shoal language. Usually when the data size is large than 1GB, that using the memory pipeline can save much of time on the file system IO between the analysis tools in the GCModeller. The upbound of the data size that in the shoal shell memory pipeline supported is up to 1TB.

  • The original R language hybrids

    • Shoal shell language can hybrids programing with R language from the plugin of RDotNET.
